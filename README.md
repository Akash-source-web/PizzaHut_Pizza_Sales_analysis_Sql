## Data Details

- Orders - orders done by users
- orders_details - details of all order 
- pizzas - contails all pizza name, price, their varients
- pizza_types - contains categories, name, ingredients

- create database in Mysql

- convert the .csv files to SQL_tables

- Analyze relationships

- perform the given task
